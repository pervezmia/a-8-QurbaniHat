# 🐄 QurbaniHat

QurbaniHat is a modern web platform that simplifies the buying and selling of livestock for Qurbani. It provides a seamless browsing experience, secure authentication, and a responsive interface to help users easily explore and manage livestock listings.

---

## 🌐 Live Website

🔗 https://a-8-qurbani-hat.vercel.app/

---

## 🚀 Features

- 🏠 Beautiful and responsive home page  
- 🎯 Eye-catching banner section  
- 📦 Multiple sections showcasing livestock and services  
- 🔍 Browse detailed livestock listings  
- 🔐 User authentication (Sign In / Sign Up)  
- ⚡ Fast and optimized performance  
- 📱 Fully responsive across all devices  

---

## 🛠️ Tech Stack

- **Frontend Framework:** Next.js  
- **Library:** React.js  
- **Styling:** Tailwind CSS  
- **UI Components:** Hero UI

## 📂 Installation & Setup

Follow these steps to run the project locally:

```bash
# Clone the repository
git clone https://github.com/pervezmia/a-8-QurbaniHat.git

# Navigate to project folder
cd qurbanihat

# Install dependencies
npm install

# Run the development server
npm run dev
