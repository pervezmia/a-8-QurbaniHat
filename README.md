# 🐄 QurbaniHat

QurbaniHat is a modern web application designed to simplify the process of buying and selling livestock for Qurbani. Built with a clean UI and smooth user experience, it helps users browse listings, explore details, and manage accounts efficiently.

---

## 🚀 Features

- 🏠 Beautiful Home Page with responsive layout
- 🎯 Eye-catching Banner section
- 📦 Multiple Sections showcasing livestock and services
- 🔐 User Authentication (Sign In / Sign Up)
- ⚡ Fast and optimized performance
- 📱 Fully responsive design for all devices

---

## 🛠️ Tech Stack

- **Frontend Framework:** Next.js
- **Library:** React.js
- **Styling:** Tailwind CSS
- **UI Components:** Hero UI
